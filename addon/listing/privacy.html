Privacy Policy

Effective date: 2026-04-21
Developer: Chinmay Nagarkar

1. Introduction

This Privacy Policy describes how EditorLLM ("the add-on", "we") handles information when you use the EditorLLM Google Workspace Add-on. By installing or using the add-on you agree to the practices described here.

2. Data we collect

EditorLLM reads the text content of your Google Docs document only while an agent is actively running in response to a menu action or sidebar request you initiate. The add-on does not perform any background scanning, does not index your documents, and does not retain document content after a request completes.

The add-on stores the following small items in your Google account's per-user properties (visible only to you and this script):

  • Your Gemini API key (if entered in Settings)
  • Your ElevenLabs API key (if entered in Settings)
  • Your preferred AI model names and TTS voice selection
  • Debug mode preference and highlight colour preference

Audio files generated by the TTS feature are saved to a folder named EditorLLM/audio inside your own Google Drive. No copy is kept anywhere else.

3. How we use your data

When you run an agent, the relevant portion of your document text is sent to the Gemini API (Google LLC) using the API key you provide. The request travels over HTTPS directly from Google's Apps Script infrastructure to the Gemini endpoint — it does not pass through any server owned by the developer of this add-on.

When you use the TTS feature, selected text is sent to the ElevenLabs API using your own ElevenLabs API key. The same direct-connection model applies: the developer never receives or stores your audio data.

The developer of this add-on does not receive, log, or retain any document content, API keys, or generated outputs.

4. Third-party services

Use of this add-on involves the following third-party APIs, each governed by their own privacy policies:

  • Gemini API — Google LLC
    https://policies.google.com/privacy
  • ElevenLabs (TTS feature only)
    https://elevenlabs.io/privacy

You are responsible for reviewing and complying with the terms and privacy policies of these services when using your own API keys with EditorLLM.

5. User controls

You can revoke EditorLLM's access to your Google account at any time:

  1. Visit myaccount.google.com/permissions
  2. Find EditorLLM in the list of connected apps.
  3. Click Remove Access.

To delete your stored API keys and preferences, open the EditorLLM Settings panel in Google Docs and clear each field before revoking access.
