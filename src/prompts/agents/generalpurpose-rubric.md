## General Purpose (@AI) instruction quality (score 0–5)

Assess whether the markdown is a **complete playbook** for replying to `@AI` threads: tone, scope, safety rails, and a **worked example**.

### Required elements
- `## Response Style` — length, voice, when to refuse or defer.
- `## Scope` — what the assistant may infer vs must ask.
- `## Sign-off` — exact closing line (`— AI Editorial Assistant`).
- `## Example Thread` — **blockquoted** fake user / assistant turns showing ideal behaviour.

### Scoring guide

| Score | Criteria |
|:-----:|----------|
| **5** | All four sections strong; example demonstrates edge case (e.g. refusal) or multi-turn nuance. |
| **4** | Example thin but present; rules still clear. |
| **3** | Missing depth in one section; still runnable. |
| **2** | Example missing or non-blockquoted; vague rules. |
| **1** | Mostly generic chatbot policy. |
| **0** | Empty, incoherent, or evaluation error. |
